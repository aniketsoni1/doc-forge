# Support

- **Questions / ideas:** open a [GitHub Discussion](https://github.com/aniketsoni1/doc-forge/discussions) or a Question issue.
- **Bugs:** open a [Bug report](https://github.com/aniketsoni1/doc-forge/issues/new/choose).
- **Security:** see `SECURITY.md` (please do not file public issues for vulnerabilities).

Before filing, run `docforge doctor` (CLI) and include its output - it reports
your environment and which generators are available.
